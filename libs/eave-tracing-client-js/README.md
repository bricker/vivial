# Eave Client Tracing

A JS file to include in any web page to collect user actions on the client.

## Dev

run file host server with
```
ruby -run -e httpd -- -p 4000 .
```

run echo server w/ 
```
node echoserv.js 
```

Then nav to [dummy site](http://localhost:4000/bestfileever.html) to test events
